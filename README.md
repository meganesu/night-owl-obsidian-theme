# Night Owl Obsidian Theme

A translation of Sarah Drasner's Night Owl VS Code theme for [Obsidian](https://obsidian.md/).

## Features

- [ ] Light mode
- [x] Dark mode
- [x] Accessible color contrast
- [ ] Works with "Andy mode" Obsidian plugin
